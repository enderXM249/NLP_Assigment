# NLP_task — English–Hindi Dataset Processing and Analysis

Assessment 1: cleaning and filtering the [`ainlpml/english-hindi`](https://huggingface.co/datasets/ainlpml/english-hindi) parallel corpus for word-count consistency between English and Hindi sentence pairs.

## Overview

The pipeline:
1. Downloads the gated `ainlpml/english-hindi` dataset from Hugging Face (`eng.txt` + `hin.txt`, 10,000 line-aligned sentence pairs).
2. Computes word counts for every English and Hindi sentence.
3. Filters to keep only pairs where **both** sentences have between 5 and 50 words.
4. Filters further to keep only pairs where the word-count difference (English − Hindi) is between **−10 and +10**.
5. Exports the cleaned dataset to Excel with five columns: English Sentence, Hindi Sentence, Word Count (English), Word Count (Hindi), Difference.

## Files

| File | Purpose |
|---|---|
| `full_pipeline.py` | End-to-end script: authenticates with Hugging Face, downloads the dataset, filters it, and writes `final_dataset.xlsx`. |
| `requirements.txt` | Python dependencies. |
| `final_dataset.xlsx` | Cleaned output (generated by the script — not committed if it contains large/private data; see Notes). |

## Setup

```bash
pip install -r requirements.txt
```

You'll also need a Hugging Face account with **approved access** to the gated `ainlpml/english-hindi` dataset, and a personal access token (Settings → Access Tokens on huggingface.co). A **Read** token type is recommended over a fine-grained one, since fine-grained tokens require manually enabling gated-repo access.

## Usage

```bash
python full_pipeline.py
```

The script will prompt for your Hugging Face token (input hidden, never stored to disk). Alternatively, set it as an environment variable beforehand:

```bash
# Windows PowerShell
$env:HF_TOKEN="hf_your_token_here"
python full_pipeline.py
```

## Notes

- The dataset is gated on Hugging Face; access must be requested and approved on the [dataset page](https://huggingface.co/datasets/ainlpml/english-hindi) before the script will run.
- Word counts are computed via whitespace tokenization (`\S+` regex), matching simple word-count conventions for both English and Hindi (Devanagari) text.
- Do not commit your Hugging Face token or any `.env` file containing it to this repository.
